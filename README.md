# PYHGOSHIFT 프로젝트 관리 대시보드

8명의 Executive와 6개 Division으로 구성된 PYHGOSHIFT AI 경영 시스템의 프로젝트 진행 현황을 관리하는 웹 대시보드입니다.

## 기능

- **홈**: 프로젝트 통계 (전체 작업, 완료, 진행 중, 대기)
- **작업 리스트**: 모든 작업을 상태별로 필터링하여 확인
- **통계**: Phase별 진행률 시각화

## 기술 스택

- Next.js 14
- React 18
- 순수 CSS (인라인 스타일링)

## 설치 및 실행

```bash
npm install
npm run dev
```

`http://localhost:3000`에서 개발 서버가 실행됩니다.

## 빌드 및 배포

```bash
npm run build
npm start
```
